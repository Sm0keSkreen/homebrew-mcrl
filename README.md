# homebrew-mcrl

Homebrew tap for [mcrl](https://github.com/Sm0keSkreen/mcrl), the JVM agent that lifts
Minecraft's account chat restriction.

```
brew install Sm0keSkreen/mcrl/mcrl
```

`brew upgrade mcrl` picks up new releases; the installed jar's path never changes
across upgrades, so no reconfiguration is needed after upgrading. See the [mcrl
README](https://github.com/Sm0keSkreen/mcrl#readme) for the one-time setup that points
`JDK_JAVA_OPTIONS` at the installed jar.
